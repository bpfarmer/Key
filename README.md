##key: share freely##

####Running Locally####

- Install CocoaPods
- Run `$ pod install`
- Open the Key.xcworkspace project in XCode
- Get the Locke repository
- In the Locke respository:
	 + `$ gem install bundler`
	 + `$ bundle install`
	 + `$ bundle exec rake db:create db:migrate`
	 + `$ shotgun app.rb`
- Build and run the Key project
